# Atividade Prática: Seleção de Atividades

## Objetivo
Executar manualmente o algoritmo guloso da seleção de atividades e comparar o resultado com a solução ótima.

## Instância proposta
| Atividade | Início | Fim |
| --- | ---: | ---: |
| A1 | 0 | 3 |
| A2 | 1 | 4 |
| A3 | 3 | 5 |
| A4 | 0 | 7 |
| A5 | 5 | 6 |
| A6 | 5 | 9 |
| A7 | 6 | 10 |
| A8 | 8 | 11 |
| A9 | 8 | 12 |
| A10 | 11 | 14 |

## Roteiro
1. Ordene as atividades pelo horário de término.
2. Escolha a atividade que termina primeiro.
3. Elimine as atividades que conflitam com a escolhida.
4. Repita o processo até não haver mais atividades compatíveis.
5. Conte quantas atividades foram selecionadas.
6. Compare o resultado com a solução calculada pelo algoritmo de programação dinâmica.

## Perguntas para discussão
1. Por que escolher a atividade que termina mais cedo ajuda a abrir espaço para as próximas?
2. Em quais problemas um algoritmo guloso pode falhar mesmo parecendo uma boa ideia?
3. O que a programação dinâmica faz de diferente para garantir a solução ótima?
